# marsala-walking-tours
A proof of concept web application regarding a platform for creating and booking tours in the city of Marsala.

Developed as part of the exam "Introduction to Web Applications" for my bachelor degree in Computer Engineering @ Politecnico di Torino.

Developed in HTML5, CSS3 and Jinja using Bootstrap for the frontend, with Python and Flask for the backend.

The application contains some sample data, between tours, guide accounts, participant accounts and a platform administrator account.

## Testing instructions
Open a terminal in the repository, make sure that Python and the requirements listed in *requirements.txt*, run the command *flask run* and (with a web browser) connect to *localhost:5000* (or as listed by the output of the flask run command)
